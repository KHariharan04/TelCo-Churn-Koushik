What the project does 

Analysis of the Telco Customer Churn Dataset, Exploratory data analysis and Machine learning and deep learning models to predict the probability of 'churning'


The goal - Predict churn for the TelCo Dataset



How to access the dataset?

The dataset can be downloaded from the link below:

https://www.kaggle.com/datasets/blastchar/telco-customer-churn

Download the dataset to a particular folder

Then, in RStudio, go to Session -> Set working directory -> And set the working directory to where the file is downloaded

After doing this, the code is ready to run.



The files submitted herewith on GitHub are:

Assessment2_Code.R (the code for the project)

readme.txt (this document)

WA_Fn-UseC_-Telco-Customer-Churn.csv - The TelCo Customer Churn dataset, for full reproducibility
